# Diamond Price Analysis Project

This repository contains a complete data analysis assignment based on the Diamonds dataset.

## Files
- `diamonds.csv` - dataset used for the analysis
- `Diamond_Price_Analysis_Project.ipynb` - Jupyter Notebook
- `Perfect_Final_Diamond_Project.pdf` - final PDF report

## Project Summary
The project analyzes how diamond characteristics such as carat, cut, color, clarity, and dimensions affect price. It includes data cleaning, descriptive statistics, visualizations, and a simple prediction model using Linear Regression.

## Dataset
- **Name:** Diamonds dataset
- **Source:** Seaborn sample dataset repository
- **Original shape:** 53,940 rows x 10 columns
- **Cleaned shape:** 53,794 rows x 10 columns

## How to Run
1. Open the notebook in Jupyter Notebook or Google Colab.
2. Keep `diamonds.csv` in the same folder as the notebook.
3. Run all cells from top to bottom.
